# Problem

Find out all LCS of《ABACABAC》and《CABCBAAB》.

# Solution

## Example

解LCS所設計的遞迴公式：

```c
c[i, j] = 0 if i = 0 or j = 0 
        = c[i-1, j-1] + 1 if i, j > 0 and x_𝑖 = y_𝑖
        = max(c[i, j-1], c[i-1, j]) if i, j > 0 and x_i ≠ y_j
```
